# Help Images

This folder stores hover-help images used in the Post-capture manual filter form.

Structure:

- `carrier_host/` - help images for leg `carrier -> host`
- `host_carrier/` - help images for leg `host -> carrier`
- `host_core/` - help images for leg `host -> core`
- `core_host/` - help images for leg `core -> host`

Each folder currently contains `test.svg` as a placeholder image.
